Cria um novo vértice com um identificador especificado.

Esta função aloca memória para um novo vértice e inicializa seus campos. Se a alocação de memória falhar, a função retorna NULL.

Parâmetros
idO identificador do vértice a ser criado.
valorO valor do vértice a ser criado.
Retorna
Apontador para o novo vértice criado, ou NULL se a alocação de memória falhar.

◆ EliminarAdjTodosVertices()

Vertice * EliminarAdjTodosVertices ( Vertice * vertices,
int idVertice,
bool * res )

Elimina todas as adjacências de um vértice com um determinado identificador em todos os vértices do grafo.

Esta função percorre todos os vértices do grafo e elimina a adjacência com um vértice específico, identificado pelo identificador idVertice. Se a eliminação for bem-sucedida em todos os vértices, o indicador de sucesso é definido como true.

Parâmetros
verticesApontador para a cabeça da lista de vértices.
idVerticeO identificador do vértice cujas adjacências devem ser eliminadas.
resApontador para uma variável booleana que indica o sucesso da eliminação das adjacências. Será definido como true se todas as adjacências forem eliminadas com sucesso, caso contrário false.
Retorna
Apontador para a cabeça atualizada da lista de vértices após a eliminação das adjacências.

bool * res )

Elimina um vértice com um determinado identificador da lista de vértices do grafo.

Esta função procura um vértice com o identificador especificado na lista de vértices do grafo e o elimina, mantendo a integridade da estrutura do grafo. Se o vértice for encontrado e eliminado com sucesso, o indicador de sucesso é atualizado para true.

Parâmetros
verticesApontador para a cabeça da lista de vértices do grafo.
idVerticeO identificador do vértice a ser eliminado.
resApontador para uma variável booleana que indica o sucesso da eliminação. Será definido como true se a eliminação for bem-sucedida, caso contrário false.
Retorna
Apontador para a cabeça atualizada da lista de vértices do grafo.

◆ InserirVertice()

Vertice * novoVertice,

Insere um novo vértice na lista de vértices do grafo.

Esta função insere um novo vértice na lista de vértices do grafo, mantendo a ordem crescente dos identificadores dos vértices. Se a lista estiver vazia, o novo vértice é inserido como o primeiro vértice. A função também atualiza um indicador de sucesso.

Parâmetros
verticesApontador para o início da lista de vértices.
novoVerticeApontador para o novo vértice a ser inserido.
resApontador para uma variável booleana que indica o sucesso da inserção. Será definido como true se a inserção for bem-sucedida, caso contrário false.
Retorna
Apontador para o início atualizada da lista de vértices.

◆ MostrarGrafo()

void MostrarGrafo ( Vertice * grafo)

Mostra os vértices e suas adjacências no grafo.

Esta função percorre a lista de vértices do grafo e imprime cada vértice com os seus adjacentes, juntamente com os pesos das adjacências. Os vértices são mostrados na ordem em que aparecem na lista.

Parâmetros
grafoApontador para o vértice inicial da lista de vértices do grafo.
