Função para atualizar um resultado com base em uma nova soma e caminho.

Esta função atualiza um resultado com base em uma nova soma e caminho encontrados durante a análise de um grafo. Se a soma atual for maior que a soma armazenada no resultado, a soma e o caminho no resultado são atualizados com os valores fornecidos.

Parâmetros
resO apontador para o resultado a ser atualizado.
somaAtualA soma atual a ser comparada com a soma armazenada no resultado.
caminhoAtualO apontador para o vértice representando o caminho atual a ser armazenado no resultado.
Retorna
Retorna 0 se a atualização for bem-sucedida, caso contrário, retorna -1.

◆ BuscarCaminhoMaximo()

int BuscarCaminhoMaximo ( Vertice * v,
int somaAtual,
Resultado * res )

Função para buscar o caminho máximo em um grafo a partir de um vértice dado.

Esta função realiza uma busca em profundidade a partir de um vértice em um grafo, buscando encontrar o caminho máximo (com a maior soma de valores dos vértices) possível. Ela marca os vértices visitados durante a busca, atualiza o resultado com o caminho máximo encontrado e continua a busca pelos vértices adjacentes não visitados. Após a conclusão da busca em profundidade a partir de um vértice, ele é desmarcado como visitado para permitir outras buscas.

Parâmetros
vO vértice inicial da busca.
somaAtualA soma acumulada dos valores dos vértices ao longo do caminho atual.
resO apontador para o resultado onde o caminho máximo será armazenado.
Retorna
Retorna 0 após concluir a busca em profundidade a partir do vértice dado. Retorna -1 se o vértice dado for nulo.

bool BuscarProfundidade ( Vertice * v,
int idDestino,
bool * visitado,
int * soma,
Grafo * g )

Função para realizar uma busca em profundidade num grafo a partir de um vértice dado.

Esta função implementa uma busca em profundidade num grafo, começando a partir de um vértice inicial e procurando por um vértice de destino. Ela marca os vértices visitados, acumula a soma dos valores dos vértices visitados e verifica se o vértice de destino é alcançável a partir do vértice inicial.

Parâmetros
vO vértice inicial da busca.
idDestinoO identificador do vértice de destino a ser encontrado.
visitadoUm array de booleanos indicando se cada vértice foi visitado ou não.
somaUm apontador para a soma acumulada dos valores dos vértices visitados.
gO apontador para o grafo onde a busca será realizada.
Retorna
Retorna verdadeiro se o vértice de destino for encontrado a partir do vértice inicial, caso contrário, retorna falso.

◆ CalcularSomaCaminho()

int CalcularSomaCaminho ( Grafo * g,
int idOrigem,
int idDestino )

Função para calcular a soma dos valores ao longo do caminho entre dois vértices num grafo.

Esta função calcula a soma dos valores dos vértices ao longo do caminho entre dois vértices num grafo utilizando uma busca em profundidade. Ela verifica se os vértices de origem e destino existem no grafo, marca os vértices visitados durante a busca, realiza a busca em profundidade para encontrar o caminho entre os vértices de origem e destino e retorna a soma dos valores dos vértices no caminho encontrado.

Parâmetros
gO apontador para o grafo onde a busca será realizada.
idOrigemO identificador do vértice de origem.
idDestinoO identificador do vértice de destino.
Retorna
Retorna a soma dos valores dos vértices no caminho entre os vértices de origem e destino. Retorna -1 se um ou ambos os vértices não existirem no grafo ou se não houver caminho entre eles.

Função para encontrar o caminho com a soma máxima de valores em um grafo.

Retorna
Retorna um apontador para o resultado contendo o caminho com a soma máxima de valores. Retorna NULL se não for possível alocar memória para o resultado.
