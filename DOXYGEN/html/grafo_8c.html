◆ InserirVerticeGrafo()

Grafo * InserirVerticeGrafo ( Grafo * g,
Vertice * v,
int * res )

Insere um novo vértice no grafo. Esta função insere um novo vértice no grafo, desde que o vértice seja válido e ainda não exista no grafo.

Parâmetros
gApontador para o grafo onde o vértice será inserido.
vApontador para o vértice a ser inserido.
resApontador para um inteiro onde será armazenado o resultado da operação:
  • 1: Inserção bem-sucedida.
  • 0: Vértice inválido, a inserção não foi realizada.
  • -1: O grafo é inválido, a inserção não foi realizada.
  • -2: O vértice já existe no grafo, a inserção não foi realizada.
Retorna
Retorna um apontador para o grafo atualizado após a inserção do vértice.
